Animations

There are 7 available animations. 3 of them are built in: shake, spin (CW and CCW) and wiggle. The other 4 (fade, hinge, slide and zoom) can be created by using the Sass mixins.

The buttons below allow you to apply an animation to the red element on the right.

Built-in animations

  • shake
  • spin-cw
  • spin-ccw
  • wiggle

Animations built using available keyframe functions

  • custom-fade
  • custom-hinge
  • custom-slide
  • custom-zoom

Custom animations

  • custom-spin-ccw
  • custom-combination
